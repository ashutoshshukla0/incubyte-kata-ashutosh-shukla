# Pokémon Frontend Engineering Kata

## Overview
Welcome to the Pokémon Frontend Engineering Kata! This project is an assessment of frontend engineering skills, focusing on craft, Test-Driven Development (TDD), and the effective leverage of AI tools to build a production-ready application.

The goal is to build a functional Pokémon application using the [PokéAPI](https://pokeapi.co/) that allows users to browse and view details about different Pokémon.
